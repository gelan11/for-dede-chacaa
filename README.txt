
Cara Ganti Playlist & Foto:
----------------------------

1. Ganti Playlist Spotify:
   - Buka index.html
   - Temukan bagian <iframe> Spotify
   - Ganti URL playlist-nya dengan yang kamu mau

2. Ganti Foto:
   - Masukkan foto kamu ke folder `images/`
   - Ubah nama file di index.html bagian <div class="gallery">

3. Upload ke GitHub:
   - Buka GitHub lewat HP
   - Upload semua isi folder ini
   - Aktifkan GitHub Pages di settings repo kamu
